# Challenge Javascript

Mode : `solo`   
Temps imparti : `5 jours` 

## Ma mission

Ajouter des graphiques:

- Ajouter deux graphiques statique au dessus des tableaux.

### Quel langage utilisés ?

- Html et css.
- Jquery
- Javascript
- Ajax and json
